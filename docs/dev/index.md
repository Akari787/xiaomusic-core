# 开发文档索引

本目录存放开发阶段文档，不作为对外 API 契约。

重要说明：

- `implementation/`、`archive/`、`network_audio/` 下的阶段性记录可能包含历史接口名与历史来源术语（如 `network_audio (deprecated)`），仅用于追溯，不作为当前推荐调用方式。
- 当前正式 API 合同始终以 `docs/api/api_v1_spec.md` 为准。

## 目录说明

- `implementation/`：重构分阶段计划、验收记录、清理报告。
- `spec/`：重构期技术规格草案与接口设计稿。
- `architecture/`：重构阶段的架构设计文档与方案草案。
- `network_audio/`：网络音频链路的开发测试文档与示例。
- `archive/`：历史回归基线和已归档文档。

## 正式 API 规范入口

- `docs/api/api_v1_spec.md`
- 认证运行时恢复稳定策略：`docs/spec/auth_runtime_recovery.md`
